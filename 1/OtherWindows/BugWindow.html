Bug Window

The Database Control Bug Window can be used to submit bugs directly to us along with information from your project, increasing the chances of us being able to reproduce the bug and hence the chances of fixing it.
It can also be used to ask us questions or send any kind of message to us such as a feature request.

You can open the window at: Top Menu Bar, Window, Database Control Pro, Other, Submit Bug or Question

At the top of the window there are two tabs 'Question' and 'Bug'. If you have found a bug select 'Bug' otherwise select 'Question'.
If you have selected 'Bug' you have some more options bellow this. If the bug was with one of the Editor Windows, select the tab with the window's name. If the window is not listed select 'Other Window Bug'. If the problem was trying to run a Command Sequence through code select 'Run Code Bug'. If the problem was not with any of these select 'Other Bug'.

Then fill in the field to give your bug/question a title. This should be short and very briefly describe what the bug/question is.

Next you need to fill in a description. This should clearly explain what the bug/question is. If it is a bug, please try to explain how we can reproduce it.

The next field is a toggle. If it is selected then you can enter your email address bellow it. We will use this to contact you when we have recieved or fixed the bug, or to answer your questions.

If you have a bug which occurs using a specific Database or Command Sequence, please select the Database or the Command Sequence in the window. If the problem is with a Command Sequence, please tell us in the description, what the Command Sequence should do and what it actually does.

Then hit the 'Send' button to send your bug or question. If you have provided your email address, we will try to reply within the next 7 days. And if you have found a genuine bug, we will try to fix it and include the fix in the next update of Database Control Pro. If the bug is server-side, then we may be able to fix it immediatley. Obviously, we have no idea how long it will take us to fix the bug and if we have lots of bugs we will solve the most severe ones first.
