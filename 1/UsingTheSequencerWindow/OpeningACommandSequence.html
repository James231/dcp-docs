Opening a Command Sequence

To Open a Command Sequence you need to open it in the Sequencer Window. This can be done in two ways:

1. Open the DCP Setup Window at the Top Menu Bar, Window, Database Control Pro, Setup Window. Find the database in the list which has the Command Sequence attached to it. Click the ‘Sequences’ button next to the Database’s name. Then you should see a list of the Command Sequences in that database. Click the ‘View/Edit’ button next to the Command Sequence you want to open and it should open it in the Sequencer Window.

2. Open the DCP Sequencer Window directly at the Top Menu Bar, Wi
ndow, Database Control Pro, Sequencer Window. In the central panel you should see a list of your databases. Select the database with the Command Sequence attached, then press the button for the Command Sequence you want to edit in the panel on the right. This should open the sequence.
